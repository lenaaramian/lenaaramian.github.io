# Personal Research Website

This is the personal research website for Lena Aramian, a researcher in finance at Stockholm Business School, Stockholm University, Sweden.

## Website Structure

The website consists of four main pages:

- **Home** (`index.html`) - About me section with contact information
- **Research** (`research.html`) - Academic research publications
- **Opinion and Policy** (`opinion_and_policy.html`) - Policy papers and opinion pieces
- **Teaching** (`teaching.html`) - Teaching experience and course information

## Files

- `index.html` - Homepage
- `research.html` - Research page
- `opinion_and_policy.html` - Opinion and Policy page
- `teaching.html` - Teaching page
- `style.css` - Main stylesheet for the homepage
- `research_style.css` - Stylesheet for research, opinion, and teaching pages
- `image/` - Directory containing profile picture

## Setup Instructions

1. Add your profile picture as `image/image.jpg` in the image directory
2. The website is ready to be deployed to GitHub Pages or any web hosting service

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Libre Baskerville, Source Sans Pro)
